# RailsWarp
> A Rails plugin for elegant, hash-based response wrapper for clean Rails API responses.

## Usage
How to use my plugin.

## Installation
Add this line to your application's Gemfile:

```ruby
gem "rails_warp"
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install rails_warp
```
